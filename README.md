# 🏏 Cricket Database Management System

A web-based application to manage cricket-related data such as players, stadiums, schedules, rankings, and more.
This system provides both **Admin** and **User** functionalities for efficient data handling.

---

## 🚀 Features

### 👨‍💼 Admin

* Add / Update / Delete players
* Manage stadium details
* Create and update match schedules
* Update rankings
* Full control over database

### 👤 User

* View player details
* Search players
* View schedules and rankings

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS
* Backend: PHP
* Database: MySQL
* Server: XAMPP

---

## 📁 Project Structure

```text
cricket-database-system/
│
├── img/                  # Images
├── index.html            # Home page
├── login / admin pages   # Authentication
├── add / delete / update # CRUD operations
├── schedule / rank       # Data modules
├── stadium / player      # Core entities
└── cricket.sql           # Database file
```

## ⚙️ Installation & Setup

### 🔹 Step 1: Install XAMPP

Download and install XAMPP

---

### 🔹 Step 2: Start Services

* Open XAMPP Control Panel
* Start:

  * Apache
  * MySQL

---

### 🔹 Step 3: Move Project Files

Copy the project folder to:

```text
C:\xampp\htdocs\
```

---

### 🔹 Step 4: Create Database

1. Open browser:

```text
http://localhost/phpmyadmin
```

2. Create database:

```text
cricket
```

---

### 🔹 Step 5: Import Database

* Go to **Import tab**
* Upload:

```text
cricket.sql
```

---

### 🔹 Step 6: Run Project

Open in browser:

```text
http://localhost/thecricket/index.html
```
